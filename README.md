# 🌌 Elemancy — 원소 연금술 조합 디펜스

원소를 모아 우주를 빚는 **원소 랜덤 조합 디펜스(원랜디)** 프로토타입.
무작위로 얻은 원소를 합성 트리대로 조합해 더 강한 타워를 만들고, **공허/엔트로피**가 보드를 무(無)로 되돌리기 전에 막아낸다.

> 순수 HTML5 Canvas + 바닐라 JS, **의존성 0**. 재미 검증용 1단계 프로토타입.

## 🕹️ 플레이 방법

1. **시작 골드 600** 으로 🎁 **랜덤 뽑기(100G)** 를 돌려 1단계 원소를 모은다 (보드에 자동 배치).
2. 보드의 **유닛을 클릭** → 그 원소가 들어가는 상위 조합이 표시된다.
   - 재료가 다 있으면 **무료 조합**, 부족하면 **🎟️ 원소 선택권**으로 부족분을 메워 조합.
3. 유닛 클릭 후 **🔀 이동**으로 위치를 바꿔 경로를 잘 커버한다.
4. 라운드마다 적이 **네모난 순환 경로**를 돈다 — 타워가 자동 공격.
5. **보드 위 몬스터가 100을 넘으면 게임오버**, **50웨이브까지 버티면 승리**.

### 라운드 흐름
- 한 라운드 **30초**. 시작 후 **15초간 30마리**가 0.5초 간격으로 출현(이때 "다음 웨이브" 버튼 잠금).
- 15초 뒤 버튼 활성 → 수동으로 다음 웨이브를 당기거나, 30초에 자동 진행.

## ⚔️ 전투 — 원소 계열별 역할

| 계열 | 공격 타입 | 주 대응 |
|---|---|---|
| 불 · 번개 | 광역 | 물량(swarm) |
| 금속 · 바람 | 단일 고화력 | 탱커(tank) |
| 물 · 얼음 | 둔화 | 고속(fast) |
| 나무 · 흙 | 버프(주변 공격력·공속↑) | 아군 강화 |

- 데미지는 **티어마다 ×10**, 적 HP는 **구간마다 ×10**으로 맞물려 "그 단계 유닛으로 약 10대"가 되도록 설계.
- 적은 5개 구간(분진기→기상기→지질기→천체기→붕괴기), 10·20·30·40·50웨이브가 **보스**.

## 🧪 합성 트리 (7단계)

| 단계 | 예시 | 규칙 |
|---|---|---|
| 1 · 기본 원소 | 물·불·바람·흙·전기·나무·얼음·금속 | 뽑기로 획득 |
| 2 | 증기(물+불), 자석(전기+금속) … | 원소 2개 |
| 3 | 구름, 번개, 마그마 … | 2단계 조합 |
| 4 | 폭풍, 화산폭발, 빙하기 … | 3·4단계 조합 |
| 5 · 천체 | 운석·혜성·가스행성·특이점 … | 4단계 천체 |
| ⭐ 전체핵 | **1단계 8원소 전부** | 6·7단계 핵심 재료 |
| 6 · 행성 | 수성~태양 | 전체핵 1 + 천체 2 |
| 7 · 최종 | 태양계·블랙홀·은하수·퀘이사 … | 전체핵 3 + 행성 2 (**게임당 1회**) |

게임 내 상단 **📜 조합식** 버튼으로 단계별 전체 트리를 볼 수 있다.

## 💰 경제

- **골드**: 라운드 시작 +200, 적 처치 소액, **보스 처치 = 보스인덱스×1000(1000~5000)**. 용도 = 뽑기 / 가챠 / 도박 / 공격력 업그레이드.
- **🎟️ 원소 선택권**: 조합 부족분을 메우는 자원. 획득 = 보스(3·6·9·12·15) + **🧭 퀘스트**(8라운드마다, 난이도 1~4 몬스터 처치 시 1~4개).
